recipients Comma seperated list of recipient labels null
destinations Comma seperated list of destination labels null
origins Comma seperated list of origin labels null
senders Comma seperated list of sender labels null
repositories Comma seperated list of repository labels null
languages Comma seperated list of languages. Options are English, French, German, or Italian null
