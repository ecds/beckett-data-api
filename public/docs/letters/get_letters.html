GET /letters

Parameters

Name Description Default
page Current page of letters 1
per_page Number of letters on a single response. 25
q Text to search. *
fields Comma seperated list of fields to be queried. recipients,mentions,destinations,origins,repositories
start_date Letters dated on or after given date. Must be in YYYY-MM-DD format. nil
end_date Letters dated on or before given date. Must be in YYYY-MM-DD format. nil
recipients Comma seperated list of recipient labels nil
destinations Comma seperated list of destination labels nil
origins Comma seperated list of origin labels nil
repositories Comma seperated list of repository labels nil
