Letters API

GET /letters?start_date=:YYYY-MM-DD - On or After Date

GET /letters

Parameters

Name Description Default
page Current page of letters 1
per_page Number of letters on a single response. 25
search Text to search. *
fields Comma seperated list of fields to be queried. recipients, mentions, destinations, origins, repositories
start_date Letters dated on or after given date. Must be in YYYY-MM-DD format. null
end_date Letters dated on or before given date. Must be in YYYY-MM-DD format. null
recipients Comma seperated list of recipient labels null
destinations Comma seperated list of destination labels null
origins Comma seperated list of origin labels null
senders Comma seperated list of sender labels null
repositories Comma seperated list of repository labels null
languages Comma seperated list of languages. Options are English, French, German, or Italian null
