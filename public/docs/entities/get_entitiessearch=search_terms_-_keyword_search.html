Entities API

GET /entities?search=:search_terms - Keyword Search

GET /entities

Returns a paginated list of entities from key word search.

Parameters

Name Description Default
page Current page of entities 1
per_page Number of entities on a single response. 25
search Text to search. *
type Limit responses by single type. Options are attendance, music, organization, person, place, production, public_event, publication, reading, translating, work_of_art, writing, and generic. null
